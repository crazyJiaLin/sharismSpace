# sharismSpace

## 项目描述 
    * 畅享阁：致力于实现互联网的信息分享平台，其中包括各类博客分享，文件、照片存储以及分享，网站收藏等功能；<br>
    * 畅享阁个人管理系统，是畅享阁的一个分支系统，作为管理畅享阁中个人信息的管理，支持对各个模块的增删改查，权限设置等；<br>
    
## 技术分享
    本项目是此分支系统的前台，采用 vue-cli + elementUI + webpack 等技术，实现整个系统单页面跳转，以及代码的优化压缩；<br>
